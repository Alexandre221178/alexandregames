Glyph Class Skill

How does it work?

Classes with glyph skills are coming to Hero Wars Alliance later this year 2023 and will make battles more competitive.

Special glyph abilities are released as soon as a Class Hero reaches level 40 in all current glyphs, after which the class glyph will be opened that will go up to level 80.

The class glyph's power becomes stronger each time a class hero's glyph set completes all glyphs at level 40. For this reason, the more heroes of a class complete glyphs, the stronger their class glyph set is. Heroes of the given class will become stronger.

If 8 out of 10 tank class heroes have full glyphs, then the power of the class ability will be 80% for all your tank class heroes.

Glyph System Update from Hero Wars Developers


Introduction

Hello all! We would like to share some interesting news with you today. A new feature, or rather, a redeveloped old feature, is on its way to Dominion! We are sharing the initial concept which might undergo some changes before the final release. Please read the document carefully till the end and let us know what you think of it! Role Simplification

Previously, each hero in Hero Wars: Alliance could have many various roles. Soon, all these roles will be transformed into classes to offer a clear understanding and concise descriptions of each hero's potential abilities. The list of classes includes: Tank, Warrior, Marksman, Mage, Healer, Support, and Control.


Class Abilities


Availability of Class Abilities: all players (no level restrictions)

equirements: level 40 of all hero’s Glyphs

Each class will also receive a passive skill called Class Ability, which can be used by any hero belonging to this class to help you in the battle.

When a specific Hero has all of their Glyphs maxed out (at level 40), the new Class Glyph will unlock, giving you access to Class Abilities. Class Glyph levels are shared among all the heroes belonging to a specific Class, and each of them contributes to its level, making Class Abilities even stronger. You can focus on one hero in the available class, or level up multiple ones instead. The new resource, Runic Spheres, will be used to increase the maximum cap of current Glyphs by 5 levels. Maximum Glyph Level cap: lvl 80.

The Highwaymen mode, available in Outland, is getting an upgrade. It will now feature a new Orange-grade opponent and a Chest of higher rarity. You will be able to get Runic Spheres from this Chest.

Additionally, there won’t be any Small, Common, Rare, or Great Enchantment Runes anymore - they all will be converted into a new resource called Runic Stones, whiсh will be used for Glyph upgrades instead. All the places and quests where you previously could acquire Runes of different rarity, such as Heroic Chest, Hero’s Way or daily quests, will receive this update as well - you will be able to get more resources for Glyph upgrades.

The List of Class Abilities

Tank Class Abilities

Class Tank
Heroes Aurora, Chabba, Mushy & Shroom, Luther, Julius, Corvus, Astaroth, Cleaver, Ziri, Rufu
Passive Skill

If your ally has less than X HP, it redirects any inflicted damage from this hero to Tank (no more than X%) for X sec. This passive skill can be triggered once per X sec for each allied hero.

Warrior Class Abilities

Class Warrior
Heroes Elmir, Yasmine, Galahad, Ishmael, Qing Mao, Tristan, K’arkh, Kayla
Passive Skill

Increases Physical Attack for X% for X sec, for each opponent affected by Warrior’s auto-attack. No more than X stacks.

Marksman Class Abilities

Class Marksman
Heroes Jhu, Dark Star, Artemis, Daredevil, Fox, Ginger, Astrid & Lucas, Keira, Dante, Cascade
Passive Skill

If the hero's target has less than X% HP left, it increases the damage inflicted by Marksman by X%. This passive skill can be triggered once per X sec.

Mage Class Abilities

Class Mage
Heroes Kai, Mojo, Cornelius, Orion, Iris, Lilith, Lars, Xe’Sha, Heidi, Satori
Passive Skill

When a Mage hero enters the field, they immediately gain X% energy at the beginning of the battle. The Mage hero gains additional X% energy for each cast skill.

Healer Class Abilities

Class Healer
Heroes Thea, Markus, Morrigan, Dorian, Aidan, Maya, Martha
Passive Skill

When any type of healing or shield is applied to an ally, it triggers an effect that transforms any incoming damage into healing for X sec (no more than X%). This passive skill can be applied to a hero once per X sec.

Support Class Abilities

Class Support
Heroes Alvanor, Fafnir, Helios, Sebastian, Jet, Nebula, Amira, Celeste
Passive Skill

Each time when Buff is applied to an ally or when damage affects enemy team, all opponents receive a debuff that lowers Armor and Magic Defence by X% for X sec. This effect can be triggered several times, up to X stacks

Control Class Abilities

Class Control
Heroes Isaac, Arachne, Judge, Phobos, Faceless, Peppy, Jorgen, Krista, Andvari, Lian
Passive Skill

The duration of all Marks, Stun, Charm, Blind and Silence effects is prolonged for X sec.

Class Glyph Example


Astaroth is a Tank hero with 5 Glyphs: Magic Attack, Armor, Health, Magic Defence, and Strength. When all Glyphs are on level 40, Class Glyph unlocks, giving you an access to Tank Class Ability. The Tank Class Ability will affect all Tank heroes you have in your roster, giving them a corresponding passive skill

You need to use Runic Spheres on maxed out Astaroth’s Glyphs to level up the Tank Class Ability. Such Rune Ascend action will add +1 to Tank Class Glyph, making the Tank Class Ability even stronger, and increasing the level cap of the chosen hero’s Glyphs by 5. Each time all of your ascended Glyphs are upgraded to their maximum, you can perform the Ascending once again using Runic Spheres (to lvl 45, 50, 55 etc.). The Tank Class Glyph will stop upgrading when you reach Glyph Level 75 on a hero of your choice
